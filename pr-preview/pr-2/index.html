EMBER .mp4 Uploader

Drag & drop .mp4 files straight into a dandiset on the DANDI Archive — no backend, no Python, everything runs in your browser.

1 · Connection

Log in at your DANDI instance, click your user initials in the top-right corner, and choose “API key”. The key is kept in this browser only and is sent solely to the API server selected above.

2 · Files

⬆️

Drop .mp4 files here or click to browse

Files are checked for MP4 integrity, checksummed, and uploaded directly to the archive.

    Uploads use the same API flow as dandi upload --validation skip (NWB validation is not performed; the app only verifies that the file is a readable MP4). The dandiset must already exist and your account must have permission to edit it.
